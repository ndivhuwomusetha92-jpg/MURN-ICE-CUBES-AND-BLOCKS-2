## Project overview
MURN ICE CUBES & BLOCKS is a supplier of ice cubes and blocks based in Tshiombo, Limpopo, South Africa. This website presents company services, gallery, staff, community work and contact/enquiry forms. It includes interactive features required by Part 2 of the POE.

## Notes about features
- *Auth system*: front-end demo stored in browser localStorage. Not secure for production.
- *Forms*: currently client-side validated and simulated. To send emails, connect forms to Formspree, EmailJS, or a backend endpoint (documented below).
- *Google Maps*: requires a valid API key and billing enabled for the project.
