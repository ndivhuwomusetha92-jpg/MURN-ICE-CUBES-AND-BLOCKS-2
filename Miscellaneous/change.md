# Murn Ice Cubes & Blocks - Final Documentation

##  Website Overview

Murn Ice Cubes & Blocks is a fully functional commercial website designed to represent a professional ice distribution company operating in Thohoyandou, Tshilamba and Louis Trichardt. The website showcases services, employees, gallery, enquiry order system, login/registration system and dynamic web functions.

The structure and content aim to reflect real business operations and meet industry-level web standards.

A fully responsive and professional business website for **Murn Ice Cubes & Blocks**, including:
- User login system (local simulation)
- Search functionality for employees and services
- Image gallery with filters
- About page with employee qualifications
- Modern UI with responsive layout and animation
- SEO-optimized metadata and structure

## Enquiry Form Logic

The enquiry page allows a user to:

1. Select product type (cubes/blocks)
2. Enter quantity
3. Validate numeric input
4. Generate an automatic quotation total
5. Display confirmation output
